LOGARITHM

Logarithm is just another way of expressing exponents and can be used to solve problems that cannot be solved using the concept of exponents only. The logarithm of any positive number, whose base is a number (>0) different from 1, is the exponent that indicates the power to which a base number is raised to produce the given number.

\[ \text{For } a,x \text{ both are positive real numbers, if } a^{m}=x \Longleftrightarrow \log_{a}x=m \] where, \[ a\neq1,\qquad a>0,\qquad x>0 \] Here,
  • a = Base
  • x = Argument (Number)
  • m = Logarithm

The short form of logarithm is log. We write it as \[ \log_a x \] and read it as log of x to the base a.

Examples

\[ 10^1=10 \Rightarrow \log_{10}10=1 \]
\[ 10^3=1000 \Rightarrow \log_{10}1000=3 \]
\[ \left(\frac1{27}\right)=3^{-3} \Rightarrow \log_3\left(\frac1{27}\right)=-3 \]
\[ \left(\frac1{49}\right)=7^{-2} \Rightarrow \log_7\left(\frac1{49}\right)=-2 \]

Fundamental Logarithmic Identity

From the definition of the logarithm of the number b to the base a, we have the identity

\[ a^{\log_a b}=b \] where \[ a>0,\qquad a\neq1,\qquad b>0 \] This is known as the Fundamental Logarithmic Identity.

PROPERTIES OF LOGARITHM

\[ \boxed{\log_a(p\times q)=\log_a p+\log_a q} \]

But

\[ \boxed{\log_a(p+q)\neq\log_a p+\log_a q} \]
Example: \[ \log_3(2\times3)=\log_3 2+\log_3 3 \]
\[ \boxed{ \log_a\left(\frac{p}{q}\right)= \log_a p-\log_a q } \]

But

\[ \boxed{ \log_a(p-q)\neq \log_a p-\log_a q } \]
Example: \[ \log_3\left(\frac23\right) = \log_3 2-\log_3 3 \]
\[ \boxed{ \log_a(p^q)=q\log_a p } \]
Example: \[ \log_3(2)^3 = 3\log_3 2 \]
\[ \boxed{ \log_c b= \frac{\log_a b} {\log_a c} } \]
Example: \[ \log_3 2 = \frac{\log_5 2} {\log_5 3} = \frac{\log_2 2} {\log_2 3} =\cdots \]
\[ \boxed{ \log_b a= \frac1{\log_a b} } \]
Example: \[ \log_5 8= \frac1{\log_8 5} \]
\[ \boxed{ \log_b a\cdot \log_c b = \log_c a } \]
Example: \[ \log_4 8 \cdot \log_2 4 = \log_2 8 \]
\[ \boxed{ \log_a a=1 } \] because \[ a=a^1 \]
Example: \[ \log_2 2=1 \]
\[ \boxed{ a^{\log_a x} = x^{\log_a a} } \] where \[ a>0,\qquad x>0 \]
Example: \[ 2^{\log_2 x} = x^{\log_2 2} \]
\[ \boxed{ \log_a1=0 } \] because \[ a^0=1 \]
Example: \[ \log_2 1=0 \]
\[ \boxed{ \log_b c = -\log_{\frac1b}c = -\log_b\left(\frac1c\right) } \]
Example: \[ \log_2 5 = -\log_{\frac12}5 = -\log_2\left(\frac15\right) \]

NOTE

  • If base of log is not given then let base of log is 10 i.e., \[ \log m=\log_{10}m \] This is called Common Logarithm.
  • If base of log is \(e\) then the log is called Natural Log or Napierian Logarithmic Function. \[ \ln x=\log_e x \]
  • Base of logarithm is never \(0\), negative or \(1\).
  • Logarithmic function is positive as well as negative but exponential function is always positive.
  • \[ \log_a0 \] is not defined.

CHARACTERISTIC AND MANTISSA

The integral part of a logarithm is called the Characteristic and the non-integral part is called the Mantissa.

\[ \log N= \text{Integer} + \text{Fractional (Decimal) Part} \]
  • Characteristic → Integral Part
  • Mantissa → Decimal Part
Example 1 \[ \log234=2.36922 \] Hence,
  • Characteristic = 2
  • Mantissa = 0.36922
Example 2 \[ \log0.234=-0.63078 \] \[ =-1+(1-0.63078) \] \[ =-1+0.36922 \] Hence,
  • Characteristic = -1
  • Mantissa = 0.36922

RULES FOR CHARACTERISTIC

(i) If \(N>1\)

The characteristic of \[ \log_{10}N \] is one less than the number of digits in the integral part of \(N\).

(ii) If \(0

The characteristic of \[ \log_{10}N \] is one greater than the number of zeros immediately after the decimal point and before the first significant digit and is a negative integer.

INSERT DECIMAL POINT IN ANTILOG

(a) Positive Characteristic

When the characteristic is \(n\), insert the decimal point after the \[ (n+1)^{th} \] digit.

(b) Negative Characteristic

When the characteristic is \(-n\), insert the decimal point such that the first significant digit is at \[ n^{th} \] place.

IMPORTANT NOTE

If characteristic of \[ \log P \] is \[ m, \] then the number of digits in \[ P \] is \[ m+1. \]

LOGARITHMIC FUNCTION

The logarithmic function is defined as

\[ f(x)=\log_a x \] where \[ a>0,\qquad a\neq1,\qquad x>0 \]
  • a → Base of logarithm
  • x → Number (Argument)

Solved Examples

1. What is the value of \( \log_{10}(0.00001) \) ?

(a) \(-5\)

(b) \(4\)

(c) \(0\)

(d) None of these

Ans. (a)
\[ 0.00001=(10)^{-5} \] \[ \log_{10}(0.00001)=\log_{10}(10^{-5}) \] \[ =-5 \] Hence, \[ \boxed{\log_{10}(0.00001)=-5} \]

2. What is the base of logarithm function for common log?

(a) None of these

(b) \(2\)

(c) \(e\)

(d) \(10\)

Ans. (d)

If base of logarithmic function is not given then we consider the base as 10. This logarithmic function is called Common Logarithm.

Ans. (c)

4. Value of \( \log_{2}64 \) is

(a) \(6\)

(b) \(5\)

(c) \(4\)

(d) \(2\)

Ans. (a)
\[ 64=2^{6} \] Therefore, \[ \log_{2}64=\log_{2}(2^{6}) \] \[ =6 \] Hence, \[ \boxed{\log_{2}64=6} \]

5. Which one is correct?

(a) \(\log(m+n)=\log m+\log n\)

(b) \(\log(m-n)=\log m-\log n\)

(c) Either (a) or (b)

(d) Neither (a) nor (b)

Ans. (d)
\[ \log(m\times n)=\log m+\log n \] but \[ \log(m+n)\neq\log m+\log n \] and \[ \log\left(\frac{m}{n}\right)=\log m-\log n \] but \[ \log(m-n)\neq\log m-\log n \] Therefore, \[ \boxed{\text{Neither (a) nor (b)}} \]

6. What is the value of

\[ \frac{\log_{11}225}{\log_{121}15} \]

(a) 2

(b) 4

(c) 1

(d) 0

Ans. (b)
\[ \frac{\log_{11}225}{\log_{121}15} = \frac{\log11^{2}}{\frac12\log15} \] Since \[ 225=15^{2} \] \[ = \frac{2\log15}{\frac12\log15} \] \[ =4 \]

7. Value of

\[ \log_{4}8 \] is

(d) None of these

Ans. (a)
Using \[ \log_ba=\frac1{\log_ab} \] \[ \log_{4}8 = \frac1{\log_{8}4} \] Hence, \[ \boxed{\log_{4}8=\frac1{\log_{8}4}} \]

8. Value of

\[ \log_{2}0 \] is

(a) defined

(b) not defined

(c) zero

(d) one

Ans. (b)
\[ \log_a0 \] is never defined because there is no real value of \(x\) satisfying \[ a^x=0. \] Therefore, \[ \boxed{\log_{2}0\text{ is not defined}.} \]

9. If \( \log 8 = x \), then value of \( \log 4 \) is

(a) \(3x/2\)

(b) \(2x\)

(c) \(2x/3\)

(d) \(3x\)

Ans. (c)
\[ \log 8=x \] \[ \log(2^3)=x \] \[ 3\log2=x \] \[ \log2=\frac{x}{3} \] Now, \[ \log4=\log(2^2) \] \[ =2\log2 \] \[ =2\left(\frac{x}{3}\right) \] \[ =\frac{2x}{3} \]
